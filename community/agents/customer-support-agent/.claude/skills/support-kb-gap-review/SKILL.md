# Support KB Gap Review

Find repeated questions, unclear docs, macro gaps, bugs, feature requests, and escalation patterns. Route docs to `knowledge-base-librarian`, product/project issues to `project-manager-agent`, and sales-related questions to `sales-followup-agent`.
