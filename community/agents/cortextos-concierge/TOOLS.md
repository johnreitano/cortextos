# Tools

The Concierge discovers tools rather than assuming them.

Suggested defaults:

- Telegram for user conversation.
- cortextOS bus for tasks, approvals, crons, memory, outputs, and agent messages.
- `gog` for Google Workspace if present.
- `gh` for GitHub if present.
- `agent-browser` for browser-based setup/testing.
- KB commands for searchable memory when configured.
- Connector/MCP tools where the user has installed them.

Never store secrets here.
