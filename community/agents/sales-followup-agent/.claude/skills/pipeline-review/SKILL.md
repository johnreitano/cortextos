# Pipeline Review

1. Read local/external CRM.
2. Find stale leads by stage cadence.
3. Identify leads needing research, proposal, scheduling, or human decision.
4. Draft follow-ups but do not send without approval.
5. Route handoffs and create tasks.
6. Save `sales/reports/YYYY-MM-DD-pipeline.md`.
