# Tools

Configured during `/setup`.

Tool-agnostic options:

- Documents/files: Google Drive/gogcli, Notion, Obsidian, local folders, SharePoint, Dropbox.
- Video/audio: transcripts, `yt-dlp`, meeting exports, local files.
- Web: `agent-browser`, exports, APIs.
- Repos: GitHub CLI/app, local clones.
- Search/storage: cortextOS KB, ChromaDB, local markdown, vector DBs, SQL stores.

Never store secrets here.
